| Файл | Какое фото |
|------|-----------|
| `hero-1.jpg` | Ванная с зелёными плитками + деревянный шкаф |
| `hero-2.jpg` | Кухня белые шкафы + деревянный остров |
| `hero-3.jpg` | Ванная большие плитки + дождливое окно |
| `hero-4.jpg` | Кухня тёмно-зелёные шкафы |
| `service-bathroom.jpg` | Лучшее фото ванной |
| `service-kitchen.jpg` | Лучшее фото кухни |
| `service-fence.jpg` | Фото забора |
| `gallery-bathroom-1.jpg` | Ванная 1 (зелёные плитки) |
| `gallery-bathroom-2.jpg` | Ванная 2 (большие плитки) |
| `gallery-bathroom-3.jpg` | Ванная 3 (двойная тумба) |
| `gallery-kitchen-1.jpg` | Кухня 1 (белые шкафы) |
| `gallery-kitchen-2.jpg` | Кухня 2 (ореховые шкафы) |
| `gallery-kitchen-3.jpg` | Кухня 3 (зелёные шкафы) |
| `gallery-fence-1.jpg` | Забор кедровый |
